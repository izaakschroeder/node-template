Overview

Template is a simple, full-featured templating system based on CSS selectors. It is capable of running both in the browser and in node.

Installation is straightforward:

Node.JS

$ npm install template

Browser

<script type="text/javascript" src="template.js"/>
Important notes:
  • Templates can only have ONE root element.
  • Your template's root element should contain xmlns="http://www.w3.org/1999/xhtml" (certain features may not work without this).
